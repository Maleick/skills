# Handoff Template

Use this handoff package at every transition to `review` or after a blocked/failure state.

## summary
- One paragraph describing what changed and why.

## changed_paths
- List modified paths, one per line.

## verification_commands
- Command
- Exit status
- Evidence

## risk_notes
- Residual risks, unknowns, and confidence level.

## next_owner
- Role and owner identifier.

## next_action
- Concrete next step with expected completion criteria.
